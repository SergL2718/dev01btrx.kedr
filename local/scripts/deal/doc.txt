1. Подготовить файл orders.csv или orders.txt.
2. Формат файла указан в файле-примере: https://megre.ru/local/scripts/deal/orders.example.txt.
3. Добавить файл orders.{csv|txt} в /local/scripts/deal/.
4. Запустить скрипт update перейдя по ссылке https://megre.ru/local/scripts/deal/update.php
5. Доступ к скрипту разрещён только администраторам сайта. То есть, перед началом работы, необходимо авторизоваться на сайте.
6. Если элементов в orders.{csv|txt} больше, чем 50 шт., тогда скрипт нужно выполнить несколько раз. Обработка не более 50 шт. - это ограничение Битрикс24.
7. Каждая строка в файле соответствует одному элементу списка.
8. Данные в элементах разделяются между собой точкой с запятой.
9. Скрипт берет данные из orders.{csv|txt}, отправляет их в Битрикс24, и после успешной отправки, удаляет обработанные данные из файла. Если в orders.{csv|txt} не осталось данных, файл будет удален.
